About the Pioneer design system

Pioneer is an un-official design system that provides Scout groups an easy way to develop websites to promote their group, sections and other information.

It is designed with UK Scout Movement's brand in mind, which was launched in 2018, and will tie in with other promotional materials.

The design system was created by Stuart Holmes after volunteering to develop a website for 4th Morpeth Scout Group, and will enable Scout groups to create their own websites, and it could be used with digital maker badges.

Pioneer is open source and can be downloaded from, or contributed to, via Github at https://github.com/trubru/Pioneer. Documentation will be added in time.

Components

Logo

You can create your logo at the Scout Brand Center - Look for the Logo Linear - Purple and customise accordingly to your group.

Replace group-logo.png with your logo file of the same name in the images folder.

Details Panel

The details panel is used to create a panel that reveals it's content when clicked.

There is a default style, and one for each type of section, so it is ideal for creating section information..
